Coisas que você precisa saber sobre o JavaScript (JS)

A linguagem JS é case sensitive (faz diferença entre maiúscula e minúscula)

No JS o ";" é opcional

Não faz diferença usar aspas ou aspas simples, só não é permitido misturar
(Dica: É recomendado que se use as aspas para "textos" e as aspas simpes para 'importações')

Variávaeis na lingaugem JS são dinâmicamente tipadas.
- String
- Number
- Boolean
- Function
- Object
-
-

Para criar uma constante usamos: const
Para criar uma variável podemos usar var ou let
var (obsoleto e inseguro (redeclarar a variável))
usar sempre o let

Concatenação (União de textos Variávaeis)
+ (operador de soma) !EVITAR (tipagem dinâmica)
Usar crase (forma moderna e segura)
`texto ${variável}`
